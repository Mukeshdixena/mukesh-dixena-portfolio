# **Mukesh Dixena – Portfolio Website**

This repository contains the source code for my personal portfolio website. The site is designed to showcase my professional experience, projects, skills, and achievements through a clean, responsive, and modern interface. It is built using Vue.js and Vite, ensuring fast performance and a streamlined development workflow.

---

## **Key Features**

* Responsive layout optimized for desktop and mobile
* Light and dark theme support
* Smooth animations using AOS
* Project showcase with structured content
* Experience, education, and achievements sections
* SEO-friendly metadata configuration
* Fast builds and hot-reload via Vite
* Deployed using GitHub Pages

## **Project Setup**

### **Clone the Repository**

```bash
git clone https://github.com/Mukeshdixena/mukesh-dixena-portfolio.git
cd mukesh-dixena-portfolio
```

### **Install Dependencies**

```bash
npm install
```

### **Start Development Server**

```bash
npm run dev
```

Development runs at:

```
http://localhost:5173/
```

### **Build for Production**

```bash
npm run build
```

### **Preview Production Build**

```bash
npm run preview
```

---

## **Deployment**

This portfolio is deployed via GitHub Pages.

If redeploying manually:

1. Build the project

   ```bash
   npm run build
   ```
2. Push the generated `dist/` folder to the `gh-pages` branch.

Your site is available at:
**[https://mukeshdixena.github.io/mukesh-dixena-portfolio](https://mukeshdixena.github.io/mukesh-dixena-portfolio)**
